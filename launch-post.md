I made high-converting-ui.

There are a hundred skills that make your UI look pretty. None of them make it *sell*. This one works on the layer that actually moves the number: the persuasion psychology underneath the pixels.

The core idea: every element on a screen asks the user a question. "Is this worth $19/mo?" is a hard question — people hesitate. "Can I try it free?" is easy — people act. Same product, same price. One framing converts multiples better. The skill teaches your AI to spot the hard questions and rewrite them into easy ones.

It ships 26 named principles (anchoring, loss aversion, goal gradient, reciprocity, smart defaults, halo effect…), each with the research behind it and the exact UI pattern that applies it — plus ordered playbooks for product pages, paywalls, pricing, onboarding, booking, and forms, and a conversion audit checklist.

Try it when you need to:
- Critique a pricing or checkout page that gets traffic but no sales
- Redesign a paywall so it asks an easier question
- Audit an onboarding flow for the doubts that make people bounce

Example:
Input: "Get access to 1,000 games — $19/mo" + a decorative illustration.
Output: "Start my free trial", a trial timeline with a pre-charge reminder for trust, real game art instead of decoration, and "no charge today, cancel anytime" under the button. Same offer — a far easier yes.

Grounded in named research (jam study, Kahneman's loss aversion, Cialdini's reciprocity), never gut-feel. Every recommendation is rule + reason + pattern.
